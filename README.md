# timespec_utils

C11構造体 `struct timespec`による処理時間計測のサンプル

## Sample

512x512 正方行列の行列積の実行時間（10回合計、平均）を測定するサンプル

```
$ gcc -std=c11 sample.c -o sample
$ ./sample
```
